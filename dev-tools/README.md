# 开发工具目录

本目录包含项目开发过程中使用的各种工具和测试文件。

## 🛠️ 工具文件

### TypeScript 工具
- **diagnose-images.ts** - 图片诊断工具
- **fix-storage-permissions.ts** - 存储权限修复工具

### HTML 测试文件
- **test-image.html** - 图片功能测试页面
- **STYLE_TEST.html** - 样式测试页面

## 💡 使用说明

这些工具主要用于：
- 开发过程中的功能测试
- 问题诊断和调试
- 权限配置验证
- UI 样式验证

**注意**: 这些文件不会被包含在生产构建中，仅用于开发环境。
